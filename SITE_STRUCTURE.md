## Component Organization

### Shared Components
- `Navbar` - Site-wide navigation
- `Footer` - Social links and copyright
- `StarsCanvas` - Animated background (contact section)

### Page-Specific Components
- `Hero` - 3D computer hero section
- `About` - Services cards
- `Experience` - Work timeline
- `Tech` - Tech stack with 3D balls
- `Works` - Project cards
- `Contact` - Contact form + 3D Earth

### 3D Canvas Components
- `ComputersCanvas` - Desktop computer model
- `BallCanvas` - Tech stack spheres
- `EarthCanvas` - Globe for contact
- `StarsCanvas` - Particle starfield

## Color Scheme (Minimal)

- **Primary Background**: Dark (#050816)
- **Secondary Background**: Dark Gray (#151030)
- **Tertiary**: Purple-tinted (#1d1836)
- **Text Primary**: White (#ffffff)
- **Text Secondary**: Gray (#aaa6c3)
- **Accent**: Pink gradient (#f272c8)
- **Gradients**: Blue, Green, Pink

## Responsive Breakpoints

- Mobile: < 640px
- Tablet: 640px - 1024px
- Desktop: > 1024px

## Interactive Elements

### Animations
- Framer Motion for smooth transitions
- Text fade-ins and slide-ups
- Card hover effects
- Smooth scrolling

### 3D Interactions
- OrbitControls on computer model
- Rotating tech stack balls
- Interactive Earth globe
- Animated particle stars
